# Ringle 기술 과제 - 안한나

## ☑️ 프로젝트 실행

#### 1. 프로젝트 클론

```bash
git clone https://github.com/ahnlook/assignment-ringle.git
```

#### 2. 패키지 설치

```bash
yarn
```

#### 3. 프로젝트 실행

```bash
yarn dev
```

#### 4. 브라우저에서 확인

[http://localhost:5173](http://localhost:5173)

## ☑️ 사용 기술

- TypeScript
- React
- Tailwind CSS
- Redux-toolkit
